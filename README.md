# Recruiterz-Django
 Recruiterz is a job posting site developed in Django. Where Employer can Register their Company profile, Login Then add Job Post. Employee can bookmark & apply for the Job. There is a dashboard section where Employer can check his job posting list & applicants details also can delete and update his job post. Employee can see his job bookmark listâ€¦

# Technology  

```
Python 3.7.10
Django 3.0
```

## Installation

```
open terminal and type
https://github.com/shahzeb-01/Recruiterz-Django.git

or simply download using the url below
https://github.com/shahzeb-01/Recruiterz-Django.git
```

## Install requirements

```
pip install -r requirements.txt
```
## Database

```
Set the database from settings.py
```

## To migrate the database open terminal in project directory and type
```
python manage.py makemigrations
python manage.py migrate
```

## Collects all static files in your apps

```
python manage.py collectstatic
```

## Run the server
```
python manage.py runserver
```
